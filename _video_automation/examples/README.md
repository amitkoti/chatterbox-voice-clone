# Video Automation Examples

This folder contains demonstration scripts, tests, and sample configurations.

## Directory Structure

### `/demos/` - Demonstration Scripts
Sample scripts showing how to use the video automation system:

- `demo_all_snowbrix_layouts.py` - Showcase all Snowbrix slide layouts
- `demo_complete_layouts.py` - Complete layout demonstrations
- `create_brand_sample.py` - Generate branded sample presentations
- `create_enhanced_samples.py` - Create enhanced presentation samples
- `create_module01_presentation.py` - Generate Module 01 presentation
- `create_snowbrix_template.py` - Create Snowbrix template presentation
- `create_snowbrix_logo_mockup.py` - Generate logo mockups

### `/tests/` - Test Scripts
Scripts for testing and verifying functionality:

- `test_alignment_fix.py` - Test slide alignment fixes
- `test_fixed_layouts.py` - Verify layout corrections
- `test_incorporated_fixes.py` - Test integrated fixes
- `test_snowbrix_snowflake.py` - Test Snowbrix/Snowflake compatibility
- `verify_fixes.py` - Verification utilities

### Files in this folder
- `sample_config.json` - Example configuration file
- `sample_presentation.pptx` - Sample PowerPoint for testing

## Usage

Run demo scripts from the project root:
```bash
python _video_automation/examples/demos/demo_all_snowbrix_layouts.py
```

Run tests:
```bash
python _video_automation/examples/tests/test_alignment_fix.py
```
